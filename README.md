This repository is a personal fork of the original [Modern Clock](https://github.com/Prayag2/kde_modernclock) by Prayag2.

Please note:

**Maintenance:** This fork was created because the original repository appears to be currently unmaintained. I am applying fixes and updates for my own use.

**Not an Official Successor:** This project is not intended to be seen as the official successor or a replacement for the original widget.

**KDE Store:** The original author still hosts this widget on the KDE Store. If you are looking for the official version, please refer to the links provided by Prayag2.

I am simply keeping this version functional for those who encounter issues with the original code. Use at your own risk.

## Changes in this fork
* Support for changing the font
* Localized text (day and month in your preferred language)
* Various minor interface tweaks

## Installation
#### From this repository
1. Clone this repository  
`git clone https://github.com/Hangman/kde_modernclock && cd kde_modernclock/`  
2. Install using the script  
`kpackagetool6 -i package`
